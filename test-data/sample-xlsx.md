# sample.xlsx 测试数据说明

用于工作流编排中 **XLSX 输入**、**数据处理**、**数据清洗**、**数据汇总** 等表格类节点的测试。

---

## 文件信息

| 项目 | 内容 |
|------|------|
| 文件名 | `sample.xlsx` |
| 建议存放路径 | `D:\Document-Intelligence-System\test-data\sample.xlsx` |
| 工作表名称 | `销售数据`（Sheet1 重命名） |

---

## 创建步骤

1. 打开 Microsoft Excel 或 WPS 表格，新建空白工作簿。
2. 将默认的 `Sheet1` 重命名为 **`销售数据`**。
3. 从 **A1** 开始，按下表填入数据（含表头共 9 行）。
4. 保存为 **`sample.xlsx`**。

## 关联测试用例

| 用例编号 | 节点 | 推荐配置 |
|----------|------|----------|
| TC-I05 | XLSX 输入 | 本地上传 sample.xlsx |
| TC-P05 | 数据处理 | 处理类型=排序，排序列=销售额，升降序=降序 |
| TC-P06 | 数据清洗 | 清洗规则=去除首尾空格 |
| TC-P08 | 数据汇总 | 分类维度=部门，指标=销售额:sum;姓名:count |
| TC-PN01 | 数据处理（负向） | 处理类型=排序，**不填**排序列 → 应报错 |

## 注意事项

- 工作流读取 Excel 时默认最多 **100 行**，本表 8 行数据在限制内。
- 节点配置中的 `sheetIndex`、`hasHeader` 当前后端**暂未实现**，按默认方式读取即可。
- 若本地上传后节点无法识别，请确认文件扩展名为 `.xlsx`，且工作表中有表头行。

## 使用方式

1. 打开应用 → **文档库** 或对话区上传对应文件  
2. **实体提取**：必须同时选 **数据源** + **Excel 模板**（如 `09` + `06`）  
3. **表格填表**：数据源 + 模板（如 `11` + `12`）  
4. **工作流**：在输入节点选文档库文件，输出节点选目标库或本地下载

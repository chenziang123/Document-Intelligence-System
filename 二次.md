# 识墨文坊 · 日常启动

> 首次安装见 `README.md`。下面假设 `.venv`、`.env`、数据库已配好。

## 0. 前置

1. 打开 **XAMPP**，启动 **MySQL**
2. 开 **两个 PowerShell** 窗口

---

## 终端 1 — 后端（8001）

```powershell
cd D:\Document-Intelligence-System
.\.venv\Scripts\Activate.ps1
$env:PYTHONPATH = "src"
cd src
..\.venv\Scripts\python.exe -m uvicorn api.main:app --reload --host 0.0.0.0 --port 8001
```

看到 `Application startup complete` 即成功。  
自检：http://127.0.0.1:8001/health

---

## 终端 2 — 前端（5174）

```powershell
cd D:\Document-Intelligence-System\extended-frontend
npm run dev
```

浏览器打开：**http://127.0.0.1:5174**

---

## 常见问题

| 现象 | 处理 |
|------|------|
| 端口被占用 | 关掉旧的后端/前端窗口；或 `netstat -ano \| findstr :8001` → `taskkill /PID <PID> /F` |
| PDF 报错缺 reportlab | 后端必须用 `.venv\Scripts\python.exe` 启动（见终端 1） |
| 前端 ECONNREFUSED | 后端没启动，先跑终端 1 |
| AI 无响应 | 检查 `.env` 里 `DEEPSEEK_API_KEY`，改后重启后端 |
